Get the client library

The client library is available on PyPy.

The client depends on some other libraries:

  • requests
  • six
  • tenacity

You can install everything like that:

    
pip install smartobjects requests six tenacity

Learn more on: GitHub.

Create a client instance

To create an instance, you can use the following python code:

    
key = "<%= clientKey %>"
secret = "<%= clientSecret %>"
url = "<%= url %>"
SmartObjectsClient(key, secret, url)
